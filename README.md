# Audiolink Script Project

VRChatのAudiolinkを活用したスクリプト開発プロジェクトです。

## プロジェクト概要

このプロジェクトは、VRChatのAudiolinkシステムを使用してインタラクティブな音楽体験を提供するスクリプトの開発を目的としています。

## ファイル構成

- `要件定義.md` - プロジェクトの要件と仕様
- `実装計画.md` - 実装計画とアーキテクチャ
- `agent.md` - エージェント設定とセットアップ

## 技術スタック

- VRChat
- Audiolink
- Udon/UdonSharp
- Unity

## 開発状況

現在、要件定義と実装計画の段階です。
